\doxysubsection{Detailed Description}
A class for the \mbox{\hyperlink{class_vivid_1_1_texture}{Texture}}. 

\mbox{\hyperlink{class_vivid_1_1_texture}{Texture}} is a class that contains the texture. It can load all formats supported by stb\+\_\+image library. The raw data of the texture is stored in the m\+\_\+\+Local\+Buffer variable.
