Contains functions to open and save files. 

\mbox{\hyperlink{namespace_file_dialogue}{File\+Dialogue}} is a namespace that contains utility functions to open and save files. This uses the native file dialogues of the OS, using the Win\+API for Windows, Cocoa for mac\+OS, and GTK for Linux.
