\doxysubsection{Detailed Description}
Contains the Transform \mbox{\hyperlink{class_vivid_1_1_component}{Component}}. 

\mbox{\hyperlink{class_vivid_1_1_transform_component}{Transform\+Component}} is a class that contains the Transform \mbox{\hyperlink{class_vivid_1_1_component}{Component}}. It contains the position, rotation, and scale of the entity. This component is a must have for every entity. It also contains the transform matrix of the entity. It also contains the gizmo operation, gizmo mode, and snap values for the gizmo.
