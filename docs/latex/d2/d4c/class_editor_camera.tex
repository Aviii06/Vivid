\doxysubsection{Detailed Description}
A class for the \mbox{\hyperlink{class_editor_camera}{Editor\+Camera}}. 

\mbox{\hyperlink{class_editor_camera}{Editor\+Camera}} is a camera that can be moved around in the scene. It can be moved forward, backward, left, right, and can be rotated using the mouse. It also has a perspective. \mbox{\hyperlink{class_editor_camera}{Editor\+Camera}} works on perspective projection and is the default camera for rendering.
