\doxysubsection{Detailed Description}
A class for the \mbox{\hyperlink{class_vivid_1_1_shader}{Shader}}. 

\mbox{\hyperlink{class_vivid_1_1_shader}{Shader}} is a class that contains the vertex and pixel shaders. It can set uniforms and bind the shader. The shader can be created by the Create function.
